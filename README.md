这是一个 Chrome 浏览器插件，还在开发中。

# 安装方式
下载 src 目录，改名为“元素图文·雅集”，然后在 Chrome 管理扩展程序里打开开发者模式加载。

# 使用说明
推荐在古文島使用这个插件，地址：https://www.gushiwen.cn/

默认为手动模式，在页面上选择标题、作者（两行文字）后，点击插件图标，选择正文，点击确认，进入浮层编辑。

可以在 Chrome 管理扩展程序、键盘快捷键里给这个插件设置一个快捷键，例如：Ctrl+Q。

在插件图标上点右键可以切换到智能选择模式，自动识别标题、作者、正文。

在浮层中双击正文区域可以编辑正文。

# 字体下载
请自行查找、下载、安装“方正金陵”字体，以获得最佳效果。
